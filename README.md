## Overview

A Python script implementing **Object-Oriented Programming (OOP)** principles to calculate geometric properties of triangles. Uses a **Shape** base class and a **Triangle** derived class to demonstrate inheritance, encapsulation, and clean class design — computing area, perimeter, type classification, and more from basic triangle parameters.

---

## Classes

### Shape (Base Class)
- Stores color and fill status
- Provides base for all geometric shapes

### Triangle (Derived Class)
- Inherits from Shape
- Calculates area (Heron's formula)
- Calculates perimeter
- Classifies triangle type (Equilateral, Isosceles, Scalene)
- Checks right triangle condition (Pythagorean theorem)
